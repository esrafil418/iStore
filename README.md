# 🛒 Simple Shopping Cart

A simple shopping cart project built with React for learning purposes.  
You can browse products and add them to your shopping cart.

---

## 🚀 Features

- View list of products
- Add products to shopping cart
- Prevent duplicate products in cart (increase quantity instead)
- Toast notifications for user actions
- Clean component structure with Context API

---

## 🛠️ Technologies Used

- **React** (v19.2.6)
- **React Icons** (v5.6.0)
- **Bootstrap** (v5.3.8)
- **FontAwesome** (v5.6.3)
